### Git 指令
在新设备或第一次使用时， 我们需要设置下git的配置：
- 告诉git当前用户名称和邮件地址

```git
git --version // 检查版本
git config --global user.name "你的用户名"
git config --global user.email "你的邮件"
```

- 设置别名，以便可以使用简洁的命令
```git
git config --global alias.st status 
// git status 相当于 git st

git config --global alias.cm commit
```
